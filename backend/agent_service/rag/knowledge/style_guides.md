# 风格指南 — 电商平台视觉风格规范

本文档描述各主流电商平台的视觉风格偏好和设计规范。

---

## Amazon 风格规范

### 核心原则
- **纯白背景**: 主图必须 RGB(255,255,255) 纯白背景，产品占比 ≥ 85%
- **无文字无水印**: 主图上不得出现任何文字、Logo、水印、边框
- **真实还原**: 图片必须真实反映产品颜色和外观，不得过度美化
- **专业摄影**: 使用专业影棚灯光，避免手机随意拍摄的质感

### 图片尺寸建议
- 主图推荐: 2000x2000px (正方形 1:1)，最小 1000x1000px
- 图片格式: JPEG（推荐）、PNG、TIFF
- 色彩模式: sRGB

### A+ Content / 高级内容
- 品牌故事模块: 可使用场景图、生活方式图，风格统一
- 功能对比表: 左侧问题/右侧方案，视觉对比清晰
- 允许: 文字叠加、图标、信息图表元素

### 视觉风格
- 干净、专业、功能性优先
- 产品主导画面，不过度依赖氛围和情感元素
- 光线: 均匀明亮，减少阴影
- 色调: 中性偏冷（增强产品真实感）

## Shopify 风格规范

### 核心原则
- **生活方式优先**: 强调产品在真实场景中的使用，建立情感连接
- **品牌故事**: 图片应传达品牌价值观和审美
- **温暖氛围**: 偏好温暖色调、自然光线
- **社交友好**: 图片适合在 Instagram/Pinterest 等平台分享

### 图片尺寸建议
- 产品图片推荐: 2048x2048px
- 集合/轮播图: 16:9 或 3:2 比例
- 支持 WebP 格式优化加载速度

### 视觉风格
- 温暖、人性化、有情感
- 场景驱动: 产品出现在使用它的真实场景中
- 色彩: 柔和、自然、品牌色融入
- 光线: 自然光、窗光、金色时段光感
- 构图: 留白充裕，不要填满画面

### 平台特性
- 支持视频缩略图和 3D 模型
- ALT 文本对 SEO 重要
- 移动端优先设计

## Taobao/Tmall 风格规范

### 核心原则
- **信息密集**: 主图通常叠加文字、促销标签、价格信息
- **视觉冲击力**: 色彩鲜明、对比度高，吸引点击
- **差异化**: 在搜索结果中脱颖而出，多用色彩对比和独特排版
- **场景+标签**: 场景图叠加促销标签是主流做法

### 图片尺寸建议
- 主图: 800x800px (1:1)，部分类目支持 3:4
- 详情页: 宽度 750px-790px，高度不限
- 手淘首页: 适配竖屏优先

### 视觉风格
- 鲜艳、有活力、高对比度
- 文字覆盖: 主图上允许文字标签、价格、促销信息
- 背景: 不限于白色，场景背景更常见
- 构图: 产品突出，画面饱满
- 色调: 温暖鲜艳，符合亚洲审美偏好

### 特殊要求
- 五张主图各有分工: 白底图、场景图、细节图、尺寸图、促销图
- 部分类目需要提供透明底 PNG

---

## JD.com 风格规范

### 核心原则
- **品质感**: 强调正品、品质，视觉上偏向高端感
- **规范严格**: 主图要求与 Amazon 类似，白色背景、无文字
- **多图展示**: 支持多张主图轮播

### 视觉风格
- 偏冷静、高级感
- 白底主图 + 场景细节图组合
- 光线: 均匀明亮，产品清晰
- 色调: 中性偏冷

---

## eBay 风格规范

### 核心原则
- **实用性**: 不需要极致的专业摄影，但要清晰展示产品
- **白底推荐**: 虽然不是强制，但白底图排名更优
- **实物拍摄**: 优先展示真实产品而非渲染图

### 图片尺寸建议
- 最小: 500x500px
- 推荐: 1600x1600px
- 免费额度内尽量使用高分辨率

## 通用电商图片设计原则

### 光线方案速查
| 产品类型 | 推荐光线方案 |
|---------|------------|
| 服装/纺织 | 大型柔光箱 + 前方45°主光 |
| 电子产品 | 顶光 + 侧光勾勒轮廓 |
| 食品 | 侧后方逆光 + 前方补光 |
| 珠宝首饰 | 多点柔光 + 微距环形灯 |
| 美妆护肤 | 环形灯 + 柔光伞 |
| 玻璃制品 | 背光 + 黑色卡纸控边 |
| 金属制品 | 大柔光箱 + 渐变背景 |

### 构图原则
1. 产品占画面 70-85% (主图)，50-70% (场景图)
2. 留白要均衡，不要随意偏置
3. 对角线构图增加动感
4. 三分法放置焦点
5. 保持呼吸感，不要画面过于拥挤

### 色调指南
- 护肤品/美妆: 清爽白色 + 柔和粉色/蓝色
- 食品/饮料: 温暖色调 + 高饱和度
- 科技/电子: 冷灰/黑色 + 蓝色强调
- 家居/生活: 温暖中性 + 木质色调
- 运动/户外: 高对比度 + 蓝色/绿色
- 奢侈品: 深色背景 + 金色强调
