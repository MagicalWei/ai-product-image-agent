# 文案模板 — 电商文案写作模板与卖点提炼技巧

本文档提供电商产品文案的写作模板、卖点提炼方法论和常见写作技巧。

---

## 卖点提炼方法论

### FAB 法则 (Feature-Advantage-Benefit)
1. **Feature (属性)**: 产品有什么？材质、工艺、规格
2. **Advantage (优势)**: 比竞品好在哪？更快、更轻、更耐用
3. **Benefit (利益)**: 给用户带来什么？省时间、省钱、更美

**示例 (保温杯)**:
- Feature: 316不锈钢内胆 + 真空双层结构
- Advantage: 比普通保温杯保温时间延长 50%
- Benefit: 早上倒的热水，下班还是烫的，全天候温暖呵护

### AIDA 模型 (Attention-Interest-Desire-Action)
1. **Attention**: 吸引注意力 — 数字/疑问/反差
2. **Interest**: 激发兴趣 — 场景化描述
3. **Desire**: 刺激欲望 — 使用后的美好体验
4. **Action**: 促成行动 — 限时/限量/紧迫感

### 4P 框架 (Problem-Promise-Prove-Push)
1. **Problem**: 指出痛点
2. **Promise**: 给出承诺
3. **Prove**: 提供证明
4. **Push**: 推动下单

## 主图文案模板

### 问题解决型
```
还在为 {痛点} 而烦恼吗？
{产品名} 采用 {核心技术}，轻松解决 {具体问题}。
{量化效果} + {使用场景}。
```

### 数字冲击型
```
{数字} 人选择 | {百分比} 好评率 | 热销 {数量}
{产品名} — {一句话定位}
{核心卖点1} · {核心卖点2} · {核心卖点3}
```

### 情感共鸣型
```
{使用场景描述，引发情感共鸣}
{产品名}，{一句话价值主张}
给 {目标用户} 的 {品类} 新选择
```

### 对比突出型
```
普通的 {品类}: {常见问题1}、{常见问题2}
我们的 {产品名}: {解决方案1}、{解决方案2}
{一句话总结优势}
```

## 标题文案模板

### 搜索优化型 (适合平台搜索)
```
{品牌词} {品类词} {核心属性} {适用场景} {差异化特点}
例: XX品牌 保温杯 316不锈钢 大容量1000ml 12小时保温 户外旅行
```

### 利益驱动型
```
{数字}+{利益点}+{品类}+{限时/促销信息}
例: 买2送1 | 防脱固发洗发水 | 90天无效退款
```

### 好奇驱动型
```
为什么 {目标用户群} 都在换 {品类}?
{产品名} 凭什么 {差异化卖点}?
```

## 五点描述模板 (Bullet Points)

适用于 Amazon 等平台的五点描述:

### 模板结构
```
1. [核心差异化卖点] — 一句话概括最大优势
2. [材质/工艺/品质] — 材料、工艺、认证
3. [功能/使用方式] — 怎么用、能做什么
4. [尺寸/规格/包含内容] — 具体参数、包装内容
5. [售后/保障] — 质保、退换政策
```

### 写作原则
- 每点首字母大写
- 控制在 200 字符以内
- 不使用促销/价格信息
- 不用 HTML 标签
- 不用主观夸张用语

## A+ Content 文案框架

### 模块化内容结构
1. **品牌故事模块**: 品牌起源 + 使命 + 差异化
2. **产品特性模块**: 左图右文，功能+利益
3. **使用场景模块**: 场景图 + 场景描述
4. **对比模块**: Before/After 对比
5. **规格参数模块**: 可视化规格表

## 场景化文案写作技巧

### 五感描述法
通过视觉、听觉、触觉、嗅觉、味觉描写让用户产生代入感:

- 视觉: 颜色、光泽、形状、体积感
- 触觉: 材质触感、温度、重量
- 听觉: 使用时的声音（如开关声、运转声）
- 嗅觉: 产品气味（如皮革、木香）
- 味觉: 食品类口感描述

### 场景公式
```
时间 + 地点 + 人物 + 痛点/需求 + 产品介入 + 美好结果
```
例: "早上7点，匆匆出门的上班族，拿起 {产品名}，一键搞定 {功能}，从容开始新的一天。"

## 不同平台的文案风格

### Amazon 风格
- 信息密度高，功能导向
- 关键词丰富（搜索优化）
- 客观、真实、可验证
- 多用数字和具体参数

### Shopify/DTC 风格
- 品牌故事驱动
- 情感连接优先
- 价值观表达
- 社交语言风格

### Taobao/Tmall 风格
- 促销感强
- 口语化、亲近感
- 强调性价比/品质感
- 多用感叹号和疑问句引起注意

### JD.com 风格
- 品质优先
- 正品保障强调
- 参数详尽
- 偏正式、可信任

## 常见文案误区

### 避免的内容
1. 虚假宣传: 夸大功效、伪造数据
2. 绝对化用语: "最好"、"第一"、"100%"
3. 竞品贬低: 点名贬低竞品
4. 无关热点: 蹭不相关的热点
5. 过度承诺: 无法兑现的承诺

### 优化方向
1. 具体 > 抽象: "12小时保温" 好于 "超长保温"
2. 利益 > 属性: "让你多睡半小时" 好于 "快速加热"
3. 场景 > 说教: "出差时不用找充电宝" 好于 "大容量电池"
4. 用户语言 > 行业术语: "不卡粉" 好于 "粉体粒径均匀"
