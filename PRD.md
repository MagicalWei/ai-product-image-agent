仓库：用户在画布当中上传了素材，每一个素材都作为一个资产，每一个单独的图片称为一个资产。添加到画布的功能应该是添加到空白的画布
最近打开：这个功能应该类似于chatbot的历史对话，打开之后可以看到用户之前在画布的编辑和与agent的对话，ai生成图之后也应该加入到仓库里面，并且做好分类。

## 视频剪辑 MVP

- 快捷工具中的“智能剪辑”使用真实 FFmpeg 任务，不使用 CSS 动画冒充视频。
- 支持多段视频上传、按秒裁剪、顺序拼接、9:16/16:9/1:1/4:5 转换、文字图层、背景音乐混流、原声音量与淡入淡出。
- Agent 只输出白名单 `VideoEditPlan`，Express 视频任务服务负责校验和执行，不接受客户端传入 Shell 命令。
- 视频任务异步执行并持久化进度；完成后生成 H.264/AAC MP4，写入当前账号仓库并关联当前会话。
- 每次进入智能剪辑都会创建可恢复的视频会话并显示在“最近打开”；保存剪辑方案、分析结果、任务状态与成片地址。
- 单文件不超过 150MB、单个视频不超过 60 秒、单任务最多 8 个源视频、成片不超过 60 秒、每账号最多 3 个排队或执行中的任务。

### 爆款结构复刻 V1

- 支持上传 1 个不超过 60 秒的参考视频，以及最多 8 个新商品图片/视频素材。
- 多模态 Agent 只拆解参考视频的钩子、节奏、镜头结构、文案公式与 CTA；不复制品牌、Logo、人物身份、完整文案或原始素材。
- Agent 输出镜头级复刻蓝图，并把每个参考镜头映射到真实商品素材；没有证据的镜头明确标记为缺失。
- 蓝图拆解期间显示环形进度与当前处理阶段；完成后再进入蓝图确认。
- 生成前必须由用户确认蓝图；图片镜头先由 Seedance 图生视频生成动态片段，再由 FFmpeg 执行 9:16 粗剪、分时字幕、音乐混流与 MP4 编码。
- 商品素材及成片进入账号仓库并异步写入 pgvector；历史素材候选按账号隔离召回并显示在蓝图中。
- 爆款结构复刻会话显示在“最近打开”，重新进入时恢复复刻蓝图、确认状态、生成任务和成片预览。

## 快捷工具箱

- 智能抠图：优先使用 `@imgly/background-removal` 本地 AI 分割，失败时降级到边缘连通算法；结果追加到当前画布并同步仓库。
- 图向反推提示词：使用 OpenAI 兼容多模态模型提取可见主体、构图、镜头、光线、色彩、背景和排版，输出可编辑、可复制、可下载的中英文 Prompt 与 Negative Prompt。
- A+/详情页：复用 `generate_product_set` Agent action 生成纵向详情图，保存对话、画布和仓库结果。
- 商品套图：复用 `generate_product_set`，支持主图、卖点图、详情图多选和并行生成。
- 爆款图复刻：复用 `style_transfer_batch`，商品图作为主体真源，参考图仅用于迁移配色、光线、构图节奏与背景语言。
